# Güvenlik Politikası

## Güvenlik Açığı Bildirme

İletiniz PHP SDK içinde bir güvenlik açığı keşfettiyseniz, lütfen **public GitHub issue açmayın**.

Bunun yerine güvenlik raporlarınızı şu adrese gönderin: **security@iletiniz.com**

Raporunuza şunları eklemeye çalışın:
- Açığın açıklaması ve etkisi
- Yeniden üretme adımları
- Etkilenen sürüm(ler)
- Mümkünse önerilen düzeltme

## Yanıt Süresi

Tüm güvenlik raporlarına **72 saat içinde** teyit yanıtı veririz. Açığın doğrulanmasının ardından düzeltme planı ve tahmini sürüm zaman çizelgesini paylaşırız.

Düzeltme yayınlandıktan sonra, izniniz olursa raporunuzu CHANGELOG'da krediteleriz.

## Desteklenen Sürümler

| Sürüm | Destek Durumu |
|---|---|
| 0.x | ✅ Aktif güvenlik desteği |
| Daha eski | ❌ Destek yok — lütfen güncelleyin |

## Açıklama Politikası

Sorumlu açıklama (coordinated disclosure) yaklaşımını izliyoruz. Düzeltme yayınlanana kadar açığı kamuya açıklamayın.
